什么是 Let's Encrypt？

Let's Encrypt 是一个免费、自动化、开放的证书颁发机构（CA），由互联网安全研究小组（ISRG）运营

完全免费

提供免费的SSL/TLS证书，没有任何隐藏费用或使用限制

自动化

通过ACME协议实现证书申请、验证和续期的完全自动化

开放标准

遵循开放的ACME协议，支持各种客户端和服务器软件

准备工作

必备条件

  • 已注册的域名（如 example.com）
  • 运行Linux的服务器（推荐Ubuntu/CentOS）
  • Root或sudo权限
  • Web服务器（Nginx、Apache等）

网络要求

  • 开放80端口（HTTP验证）
  • 域名DNS解析正确配置
  • 服务器能够访问互联网
  • 防火墙允许必要的网络流量

安装Certbot

Ubuntu/Debian系统

sudo apt update && sudo apt upgrade -y
sudo apt install certbot python3-certbot-nginx -y

CentOS/RHEL系统

sudo yum install epel-release -y
sudo yum install certbot python3-certbot-nginx -y

安装说明

Certbot是Let's Encrypt官方推荐的客户端工具，用于自动化证书的申请、安装和续期过程

申请SSL证书

支持单域名和泛域名证书申请

单域名证书

自动安装（推荐）

sudo certbot --nginx -d example.com -d www.example.com

此命令会自动申请证书并配置Nginx

仅获取证书

sudo certbot certonly --nginx -d example.com -d www.example.com

仅获取证书文件，需要手动配置Web服务器

泛域名证书

泛域名证书（如 *.example.com）可以保护主域名下的所有子域名

sudo certbot certonly --manual --preferred-challenges dns \
-d *.example.com -d example.com

DNS验证步骤

  1. 1. 执行上述命令后，Certbot会提供一个TXT记录
  2. 2. 登录您的域名管理平台
  3. 3. 添加TXT记录：_acme-challenge.example.com
  4. 4. 等待DNS记录生效（通常需要几分钟）
  5. 5. 按Enter键继续验证

证书续期

手动续期

# 测试续期
sudo certbot renew --dry-run

# 实际续期
sudo certbot renew

自动续期

# 编辑crontab
sudo crontab -e

# 添加以下行（每天凌晨2点执行）
0 2 * * * /usr/bin/certbot renew --quiet

续期说明

Let's Encrypt证书有效期为90天，建议设置自动续期以避免证书过期导致的服务中断

证书文件位置 

/etc/letsencrypt/live/your_domain/
├── fullchain.pem    # 完整证书链
├── privkey.pem      # 私钥文件
├── cert.pem         # 域名证书
└── chain.pem        # CA证书链

Nginx配置示例

server {
    listen 443 ssl;
    server_name example.com;
    
    ssl_certificate /etc/letsencrypt/live/example.com/fullchain.pem;
    ssl_certificate_key /etc/letsencrypt/live/example.com/privkey.pem;
}

Apache配置示例

<VirtualHost *:443>
    ServerName example.com
    SSLEngine on
    SSLCertificateFile /etc/letsencrypt/live/example.com/fullchain.pem
    SSLCertificateKeyFile /etc/letsencrypt/live/example.com/privkey.pem
</VirtualHost>

常见问题

解答您在使用过程中可能遇到的问题

证书有效期为什么只有90天？

90天的有效期是为了提高安全性。如果私钥泄露，攻击者只能在有限时间内使用。同时，这也鼓励用户实现自动化续期。

可以申请通配符证书吗？

可以。需要使用DNS-01验证方式，通过添加TXT记录来验证域名所有权。

证书续期会影响网站服务吗？

不会。Certbot的renew命令会检查证书是否需要续期（通常在到期前30天），续期过程不会中断正在运行的服务。

如何吊销证书？

使用命令：sudo certbot revoke --cert-path /etc/letsencrypt/live/example.com/fullchain.pem

支持哪些Web服务器？

官方支持Nginx、Apache、Haproxy等主流Web服务器，也可以手动配置其他服务器。
